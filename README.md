# SKOROZVON-GCK

[![License](https://img.shields.io/badge/License-Proprietary-red.svg)](LICENSE)

**Скорость звонков** — система автоматизации и аналитики звонков для отдела продаж.

## Возможности

- 📞 **Автоматизация звонков** — интеграция с телефонией, автоматический обзвон
- 📊 **Аналитика разговоров** — статистика по менеджерам, конверсия, длительность
- 📝 **Транскрибация** — расшифровка записей разговоров
- 🎯 **Контроль качества** — оценка менеджеров по скрипту продаж
- 📈 **Дашборды** — визуализация ключевых метрик

## Технологии

- Python / Node.js
- Интеграция с VoIP/телефонией
- База данных для хранения записей и статистики
